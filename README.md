# GuessTheNumber

Игра «Угадай число»
Описание
Игра «Угадай число» — это интерактивное приложение на Python с графическим интерфейсом, созданное с использованием библиотеки Tkinter. В игре доступны два режима:

Компьютер против Пользователя: компьютер загадывает число от 1 до 100, а игрок должен его угадать.

Пользователь против Пользователя: один игрок загадывает число, а второй пытается его отгадать.

Функциональность
Главное меню с выбором режима игры.

Плавное переключение между режимами.

Анимация наведения на кнопки (изменение цвета фона и текста).

Проверка ввода: программа обрабатывает только целые числа.

Обратная связь: сообщения о результатах угадывания («Больше», «Меньше», «Угадал!»).

Возможность вернуться в главное меню из любого режима.

Структура проекта
Проект состоит из нескольких файлов:

main.py — файл запуска приложения. Инициализирует главное окно и запускает главный цикл Tkinter.

Constants.py — файл с константами: параметры окна, цвета, тексты для элементов интерфейса.

MainMenu.py — реализует главное меню игры с кнопками выбора режима и выхода.

UserVersusComputer.py — режим «Компьютер против Пользователя»: компьютер загадывает случайное число, игрок угадывает.

UserVersusUser.py — режим «Пользователь против Пользователя»: первый игрок вводит число, второй угадывает.

Установка и запуск
Требования
Python 3.6 или выше.

Стандартная библиотека Tkinter (входит в поставку Python).

Библиотека random (встроена в Python, дополнительная установка не требуется).

Инструкция по запуску
Клонируйте репозиторий или скачайте файлы проекта в одну папку.

Убедитесь, что все файлы находятся в одной директории:

main.py

Constants.py

MainMenu.py

UserVersusComputer.py

UserVersusUser.py

Откройте терминал или командную строку и перейдите в папку с проектом.

Запустите приложение командой:

bash
python main.py
Использование
Запуск программы: после выполнения python main.py откроется главное окно игры.

Главное меню:

Нажмите «Пользователь V Компьютер», чтобы сыграть против компьютера.

Нажмите «Пользователь V Пользователь», чтобы сыграть с другим игроком.

Нажмите «Выход», чтобы закрыть приложение.

Режим «Компьютер против Пользователя»:

Введите число в поле ввода и нажмите «ОК».

Программа сообщит, больше или меньше загаданное число.

Продолжайте угадывать, пока не отгадаете число (от 1 до 100).

Режим «Пользователь против Пользователя»:

Первый игрок вводит число и нажимает «ОК» (число сохраняется в памяти программы).

Второй игрок пытается угадать число, вводя варианты и нажимая «ОК».

Программа даёт подсказки («Больше»/«Меньше»), пока число не будет угадано.

Возврат в меню: в любом режиме нажмите «Назад», чтобы вернуться в главное меню.

Файлы и их назначение
Файл	Назначение
main.py	Точка входа в приложение. Запускает главное окно (MainMenu) и главный цикл Tkinter (mainloop).
Constants.py	Хранит все константы игры: размеры окна, цвета элементов, тексты заголовков, кнопок и правил.
MainMenu.py	Реализует главное меню: заголовок, кнопки выбора режимов и выхода, анимацию наведения. Управляет переходом между режимами и главным меню.
UserVersusComputer.py	Режим игры против компьютера: генерирует случайное число (random.randint(1, 100)), проверяет ввод пользователя, выводит подсказки.
UserVersusUser.py	Режим игры двух игроков: первый игрок задаёт число, второй угадывает, программа проверяет и даёт подсказки.
