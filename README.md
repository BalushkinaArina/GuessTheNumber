# GuessTheNumber («Угадай число»)

Простая игра с графическим интерфейсом на Python (Tkinter), где можно играть против компьютера или с другим игроком.

## Обзор

Приложение предлагает два режима игры:
* **Компьютер против Пользователя**: компьютер загадывает число от 1 до 100, игрок должен его угадать.
* **Пользователь против Пользователя**: один игрок загадывает число, второй пытается его отгадать.

## Функциональные возможности


* Главное меню с выбором режима игры.
* Анимация наведения на кнопки (изменение цвета фона и текста).
* Проверка корректности ввода (только целые числа).
* Обратная связь: подсказки «Больше», «Меньше», сообщение об успехе.
* Возможность вернуться в главное меню из любого режима.
* Настраиваемые константы (цвета, тексты, размеры окна) в отдельном файле.

## Структура проекта

Угадай-число
* `main.py` - Файл запуска приложения
* `Constants.py` - Константы: цвета, тексты, размеры
* `MainMenu.py` - Главное меню игры
* `UserVersusComputer.py` - Режим «Компьютер против Пользователя»
* `UserVersusUser.py` - Режим «Пользователь против Пользователя»

## Технические требования

* Python 3.6 или выше.
* Библиотека Tkinter (встроена в Python).
* Библиотека `random` (встроена в Python, используется в режиме против компьютера).

## Установка и запуск

1. Клонируйте репозиторий или скачайте все файлы проекта в одну папку.
2. Убедитесь, что присутствуют все файлы:
   * `main.py`
   * `Constants.py`
   * `MainMenu.py`
   * `UserVersusComputer.py`
   * `UserVersusUser.py`
3. Откройте терминал или командную строку и перейдите в папку с проектом.
4. Запустите приложение командой: `python main.py`

## Настройка внешнего вида и текстов

Вы можете изменить внешний вид и тексты, отредактировав файл Constants.py:

* windowTitle — заголовок окна приложения.
* windowWidth, windowHeight — размеры окна.
* Цвета элементов: color_titleText, color_bgButton, color_hover_bgButton и т. д.
* Тексты кнопок и заголовков: labelTitleText, nameButtonModeUvC, textRules_UvC и т. д.
